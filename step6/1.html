텍스트 관련 태그들

HTML에서 텍스트를 구조화하고 스타일링하는데 사용되는 다양한 태그들을 알아보겠습니다.

제목 태그

h1 태그: 웹 페이지에서 가장 중요한 제목을 나타냅니다. 페이지당 하나만 사용하는 것이 좋습니다.

h2 태그: 주요 섹션의 제목을 나타내며, h1의 하위 제목으로 사용됩니다.

h3 태그: h2의 하위 섹션을 나타내는 제목으로 사용됩니다.

h4 태그: 세부 내용의 제목을 나타내며, h3의 하위 제목입니다.

h5 태그: 더 작은 단위의 제목을 나타내는데 사용됩니다.

h6 태그: HTML에서 제공하는 가장 낮은 수준의 제목 태그입니다.

본문 태그

p 태그: 하나의 문단을 정의하며, 앞뒤로 자동 줄바꿈이 적용됩니다.

인용 태그

q 태그: 짧은 인용문을 표시할 때 사용합니다.

blockquote 태그: 긴 인용문을 표시할 때 사용합니다.

기타 텍스트 태그

br 태그: 강제로 줄을 바꿀 때 사용하며, 닫는 태그가 없는 빈 요소입니다.

hr 태그: 문단을 구분하는 수평선을 그리며, 주제의 전환을 나타낼 때 사용합니다.

strong 태그: 텍스트를 굵게 표시하며, 중요한 내용임을 강조할 때 사용합니다.

em 태그: 텍스트를 기울여서 표시하며, 강조하고 싶은 부분을 표시할 때 사용합니다.

ins 태그: 문서에 새로 추가된 텍스트를 표시할 때 사용하며, 보통 밑줄이 그어집니다.

del 태그: 삭제된 텍스트를 표시할 때 사용하며, 보통 취소선이 그어집니다.

mark 태그: 텍스트를 하이라이트하여 강조할 때 사용합니다.

small 태그: 작은 글씨로 부가적인 정보를 표시할 때 사용합니다.

cite 태그: 작품의 제목을 표시할 때 사용합니다.
